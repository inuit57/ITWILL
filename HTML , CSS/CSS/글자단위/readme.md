# font- 로 시작하는 글자 속성

## font-size 
- 글자 크기 속성 
- 기본값은 16px 크기 
- px, em , rem 단위를 사용해서 표현 가능하다. 

| em | rem |
|----|------|
|상위요소 크기에 비례 | 최상위 요소(html) 크기에 비례 |

- 일반적으로 1em = 1rem = 16px. 
- em,rem은 주로 비율로 적용하고자 할 때 사용한다. 
- 예를 들면 0.75em , 1.25rem 


## font-weight 
- 글자 두께 
- 100~900까지 설정가능. 
- 예) font-weight : 900 ; 
- <b> font-weight: bold ;  (두껍게) </b> : 가장 많이 사용한다. 
- font-weight: normal ; (기본)
