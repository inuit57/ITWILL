# 레이아웃 태그 
- html5 에서 추가된 태그들
```
<header></header> -- 로고, 주메뉴, 로그인폼 등이 주로 위치하는 상단 영역을 지정
<nav></nav>            -- 메뉴 영역을 지정
<article></article>    -- 컨텐츠 중에서 독립형, 완성형의 성격을 띄는 영역을 지정
<section></section>    -- 컨텐츠의 주제별 구분  영역을 지정
<aside></aside>        -- 주된 컨텐츠가 아닌 영역(배너, 바로가기 등)을 지정 
<footer></footer> -- 카피라이트, 연락처 등이 주로 위치하는 하단 영역을 지정
```
