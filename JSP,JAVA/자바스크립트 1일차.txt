자바스크립트 : 
웹페이지에서 어떤 동작을 했을 때, 
그 동작에 따른 반응을 처리하는 언어 


1) 인터프리터 언어 : 코드를 위에서 아래로 순차적으로 실행 
에러 발생할 경우 더이상 실행 X 
컴파일할 필요 없이 그냥 읽는대로 바로 실행한다. 

2) 클라이언트 언어 : 사용자단의 영역을 개발하기 위한 언어
-> 서버의 부하를 감소. 서버가 다른 일을 더 할 수 있게 한다. 

3)객체기반 언어 
4)공개된 언어( 오픈소스 자료 다양) 
5)다양한 라이브러리 제공 -> Jquery 
=> HTML5 API 기반의 언어!

문자열 : '' 또는 ""로 묶어서 표현. 

str = "안녕하세요. "test"입니다. " (x)

가능한 표현들
str = '안녕하세요. "test" 입니다. ' 
str = "안녕하세요. 'test' 입니다. "
str = "안녕하세요. \"test\"입니다. " 

--
자바 스크립트는 문자형 데이터 타입으로 HTML코드를 실행가능
document.write( str + "<br>") ;
