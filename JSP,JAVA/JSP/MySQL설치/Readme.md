# 설치 과정
- Developer Default (기본 설정)
  - 나중에 익숙해지면 Custom으로 해서 필요한 것들만 설치하기
- [excute] 버튼 누르기 (필요한 것들 설치) 
  -  맨위에 MySQL Server만 제대로 설치되었다면 문제 없다. 
-  [next] 버튼 활성화 될 경우, 누르고 [Yes] -> [excute] 누르기
    - 기다리면 된다.
