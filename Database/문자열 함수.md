<h1> 문자 관련 함수 </h1>

<h2> 1) 대소문자 관련 </h2>

- Lower() : 모든 글자를 소문자로
- Upper() : 모든 글자를 대문자로
- Initcap() : 첫글자만 대문자, 나머지는 소문자인 initcap 형식으로 변경 

<h3> 1-1) 사용예시 </h3> 

~~~
select job_id, last_name 
from employees
where lower(last_name) = 'choi'; 
~~~


<h2> 2) 문자 조작 함수 </h2>

- Concat(expr1, expr2) : 두 인수를 연결해서 하나의 문자열로 반환
~~~
CONCAT('Hell' , 'o') return 'hello'
~~~
