<h1>숫자 함수</h1>

<h2> 1) ROUND ( num1, num2) </h2>
- 지정된 소수점 자릿수로 값을 <b> 반올림 </b> 해줍니다. 
  - 예) ROUND(45.926, 2) : 45.93
  - num2를 지정해주지 않는 경우 기본값 0 

<h2> 2) TRUNC( num1, num2) </h2>
- 지정된 소수점 자릿수로 값을 <b> 버림 </b> 해줍니다.
  - 예) TRUNC(45.926 , 2) : 45.92
  - num2를 지정해주지 않는 경우 기본값 0 

<h2> 3) MOD(num1, num2) </h2>
- 나눈 나머지를 반환합니다.
